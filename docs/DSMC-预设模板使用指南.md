# DSMC预设模板使用指南

## 概述

预设模板是SPARTA DSMC智能助手右侧控制面板中的一个重要功能，它为用户提供了5个精心配置的DSMC仿真场景模板，让您可以快速开始仿真而无需从零开始配置所有参数。

## 位置

预设模板选择器位于：
- **右侧控制面板** → **📋 预设模板** 部分
- 在"🚀 DSMC控制面板"的顶部

## 功能作用

### 1. **快速启动仿真**
预设模板的主要作用是让您无需手动填写复杂的DSMC参数，只需选择一个与您的仿真场景接近的模板，系统会自动填充所有必要的参数，包括：
- 维度设置（2D/3D）
- 几何形状
- 网格尺寸
- 气体类型
- 温度、压力、速度
- 大气模型
- 时间步长和仿真步数
- 碰撞模型

### 2. **学习最佳实践**
每个模板都是基于实际应用场景精心设计的，参数配置遵循SPARTA手册的最佳实践，可以作为学习参考。

### 3. **避免常见错误**
预设模板已经经过验证，避免了常见的参数配置错误，如：
- 网格尺寸过小或过大
- 时间步长与物理尺度不匹配
- 碰撞模型与气体类型不兼容
- 大气模型选择不当

## 可用模板详解

### 1. **Hypersonic Flow (Re-entry) - 高超音速再入流动**

**适用场景**：
- 航天器再入大气层模拟
- 高超音速飞行器气动分析
- 稀薄气体流动研究

**参数特点**：
- **高度**：80 km（中间层）
- **速度**：7500 m/s（~Mach 22）
- **大气模型**：NRLMSISE-00（最精确的高空大气模型）
- **温度**：196 K（高空低温）
- **压力**：1.05 Pa（极低压强）
- **网格**：150×75×75（高密度网格捕捉激波细节）
- **碰撞模型**：VSS（变软球模型，适合高温流动）

**何时使用**：
- 研究航天器返回地球时的气动加热
- 分析高超音速边界层
- 计算气动力和气动热载荷

---

### 2. **Vacuum Chamber - 真空室**

**适用场景**：
- 真空设备内的气体流动
- 低压环境中的粒子运动
- 分子泵性能分析

**参数特点**：
- **压力**：0.5 Pa（真空环境）
- **气体**：氩气（Ar，惰性气体）
- **速度**：0 m/s（静态或低速流动）
- **大气模型**：Custom（手动设置，无大气影响）
- **网格**：50×50×50（中等密度，适合封闭空间）
- **碰撞模型**：VHS（变硬球，适合低温低速）

**何时使用**：
- 模拟真空室内的气体行为
- 研究分子泵抽气过程
- 分析真空镀膜过程中的粒子输运

---

### 3. **Atmospheric Flight (0-100km) - 大气层飞行**

**适用场景**：
- 民用/军用飞行器气动设计
- 火箭上升段分析
- 探空气球轨迹模拟

**参数特点**：
- **高度**：0 km（海平面，可调整）
- **速度**：500 m/s（~Mach 1.5，亚/超音速）
- **大气模型**：NRLMSISE-00
- **温度**：288 K（15°C标准温度）
- **压力**：101325 Pa（1个大气压）
- **网格**：100×50×50（平衡精度和计算成本）
- **碰撞模型**：VSS

**何时使用**：
- 设计飞行器外形
- 分析不同高度的气动性能
- 研究跨音速/超音速流动

**提示**：可以修改`altitude_km`参数模拟不同高度（0-100 km）

---

### 4. **Shock Tube - 激波管**

**适用场景**：
- 激波物理研究
- 爆炸波传播
- 高速气体动力学实验

**参数特点**：
- **维度**：2D（简化计算，聚焦波传播）
- **网格**：200×20×1（长条形，捕捉波前细节）
- **气体**：氮气（N2）
- **速度**：0 m/s（初始静止，依靠压差驱动）
- **时间步长**：1e-7 s（极小，捕捉快速变化）
- **仿真步数**：5000（较长时间观察波的演化）
- **碰撞模型**：VSS

**何时使用**：
- 研究激波结构和强度
- 验证理论激波关系式
- 分析非定常流动现象

---

### 5. **Custom (Manual Configuration) - 手动配置** ⭐

**适用场景**：
- **所有不属于上述4种标准场景的情况**
- 需要精确控制每个参数的高级用户
- 探索性研究和参数敏感性分析

**参数特点**：
- **维度**：3D
- **几何**：立方体盒子
- **网格**：100×50×50（通用尺寸）
- **气体**：空气
- **温度**：300 K（室温）
- **压力**：101325 Pa（标准大气压）
- **速度**：1000 m/s（超音速）
- **大气模型**：**US76**（美国标准大气1976）
- **碰撞模型**：VSS

**何时使用**：
- 您的仿真场景不符合以上4种典型场景
- 需要从头开始逐一调整参数
- 想要学习DSMC参数配置

**重要说明**：
- US76是更通用的标准大气模型，适合0-1000 km高度
- NRLMSISE-00更精确但计算成本更高，适合需要高精度的场景
- Custom模板使用US76作为默认值，兼顾精度和性能

## 使用方法

### 基本使用流程

1. **打开DSMC控制面板**
   - 点击右侧的"🚀 DSMC控制面板"（如果未展开）
   - 或点击顶部的"DSMC参数设置"按钮

2. **选择预设模板**
   - 在"📋 预设模板"下拉框中选择最接近您需求的模板
   - 下方会显示该模板的简要描述

3. **查看参数**
   - 选择模板后，所有参数会自动填充到表单中
   - 您可以在AI生成输入文件前查看和微调这些参数

4. **微调参数（可选）**
   - 如果需要，可以修改个别参数以匹配您的具体需求
   - 例如：修改网格密度、调整仿真步数等

5. **生成输入文件**
   - 点击"生成"或发送消息给AI
   - AI会根据模板参数生成SPARTA输入文件

6. **运行仿真**
   - 点击"▶️ 运行仿真"按钮
   - 在日志监控区域查看实时进度

### 高级用法

#### 组合使用模板

可以先选择一个接近的模板作为起点，然后修改部分参数：

**示例1：基于"Atmospheric Flight"修改**
```
选择：Atmospheric Flight
修改：
  - altitude_km: 0 → 30（平流层）
  - velocity: 500 → 800（Mach 2+）
  - grid_size: [100,50,50] → [150,75,75]（更精细）
```

**示例2：基于"Custom"创建新场景**
```
选择：Custom
修改：
  - geometry: box → cylinder（圆柱形物体）
  - gas: Air → He（改用氦气）
  - temperature: 300 → 77（液氮温度）
```

#### 参数优先级理解

- **模板参数** < **用户手动修改** < **AI智能调整**
- AI会尊重您的手动修改，只调整未明确指定的参数

## 常见问题 (FAQ)

### Q1: 为什么需要预设模板？

**A**: DSMC仿真涉及10多个相互关联的参数，错误的参数组合会导致：
- 仿真失败（SPARTA报错）
- 结果不准确（物理不合理）
- 浪费计算资源（网格过密或时间步长过小）

预设模板确保参数组合的有效性，节省您的配置时间。

### Q2: 模板之间的主要区别是什么？

**A**: 主要区别在于**物理场景**和**参数尺度**：

| 特性 | Hypersonic | Vacuum | Atmospheric | Shock Tube | Custom |
|------|------------|--------|-------------|------------|--------|
| 压力范围 | 极低(~1 Pa) | 低(~0.5 Pa) | 标准(101 kPa) | 标准 | 标准 |
| 速度范围 | 极高(7.5 km/s) | 静态 | 中等(500 m/s) | 波驱动 | 可调 |
| 大气模型 | NRLMSISE-00 | 无 | NRLMSISE-00 | 无 | **US76** |
| 应用领域 | 航天 | 真空技术 | 航空 | 科研 | 通用 |

### Q3: 我可以保存自己的模板吗？

**A**: 当前版本不支持用户自定义模板保存。但您可以：
1. 使用Custom模板作为起点
2. 导出生成的输入文件保存
3. 下次上传该文件作为参考

**计划功能**（未来版本）：支持保存和分享用户自定义模板。

### Q4: 选错模板会怎样？

**A**: 不会造成严重后果，但可能：
- 参数不适合您的场景，需要大量手动调整
- AI生成的输入文件偏离预期
- 仿真结果不准确

**建议**：如果不确定，选择"Custom"然后详细描述您的需求给AI。

### Q5: 为什么Custom模板使用US76而不是NRLMSISE-00？

**A**:
- **US76**（美国标准大气1976）是更通用的标准大气模型
  - 适用范围：0-1000 km
  - 计算速度快
  - 适合大多数工程应用

- **NRLMSISE-00**是更精确的经验模型
  - 适用范围：0-500 km
  - 考虑太阳活动、地磁等因素
  - 计算成本更高
  - 适合需要高精度的航天应用

对于通用的Custom模板，US76提供了更好的性能/精度平衡。如果您的场景需要NRLMSISE-00，可以手动修改或选择Hypersonic/Atmospheric模板。

### Q6: 能否在运行时切换模板？

**A**:
- **仿真运行前**：可以随时切换，参数会立即更新
- **仿真运行中**：无法切换，需要停止当前仿真
- **仿真完成后**：可以切换并创建新的版本迭代

## 最佳实践建议

### 选择模板的决策树

```
您的仿真场景是？
│
├─ 高超音速飞行（Mach > 5，高度 > 50km）
│  └─ 选择: Hypersonic Flow
│
├─ 真空或极低压环境（P < 10 Pa）
│  └─ 选择: Vacuum Chamber
│
├─ 常规飞行（高度 0-100km，Mach < 5）
│  └─ 选择: Atmospheric Flight
│
├─ 激波/爆炸/快速压缩
│  └─ 选择: Shock Tube
│
└─ 其他/不确定/需要完全自定义
   └─ 选择: Custom
```

### 参数调整建议

选择模板后，通常需要调整的参数：

**必调参数**（根据您的具体场景）：
- `altitude_km`（如果使用大气模型）
- `velocity`（来流速度）
- `num_steps`（仿真时长）

**可选调整**（优化性能或精度）：
- `grid_size`（网格密度，影响精度和速度）
- `timestep`（时间步长，通常自动计算）
- `collision_model`（碰撞模型，一般不需要改）

**不建议改动**（除非您是专家）：
- `dimension`（改变维度会大幅影响计算成本）
- `geometry`（改变几何需要重新设计网格）

## 技术原理

### 模板如何工作？

1. **存储**：所有模板存储在 `llm-chat-app/static/data/dsmc-templates.json`
2. **加载**：页面加载时，JS读取模板文件并填充下拉框
3. **选择**：用户选择模板时，`loadTemplate()`函数被调用
4. **应用**：模板参数覆盖表单的默认值
5. **传递**：用户点击"生成"后，参数传递给AI生成输入文件

### 模板数据结构

```json
{
  "id": "模板唯一标识符",
  "name": "模板显示名称",
  "description": "简要描述",
  "parameters": {
    "dimension": "2d或3d",
    "geometry": "几何形状",
    "grid_size": [nx, ny, nz],
    "gas": "气体类型",
    "temperature": 温度(K),
    "pressure": 压强(Pa),
    "velocity": 速度(m/s),
    "altitude_km": 高度(km)或null,
    "atmospheric_model": "大气模型名称",
    "timestep": 时间步长(s),
    "num_steps": 仿真步数,
    "collision_model": "碰撞模型"
  }
}
```
